# OCR Переводчик с экрана

## 📖 Описание
Приложение для автоматического перевода текста с экрана в реальном времени через API Яндекс.Переводчика. Перевод происходит по нажатию клавиши 'T'.

## 🚀 Особенности
- Захват текста с экрана через OCR
- Перевод через Яндекс.Переводчик API
- Управление по горячим клавишам
- Графический интерфейс

## ⚙️ Установка

### Для запуска из исходников:
1. Установите Python 3.8+
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
3. Установите Tesseract OCR
4. Создайте сервисный аккаунт в Яндекс.Облаке и получите key.json
5. Замените FOLDER_ID в коде на ваш ID каталога

## Для запуска exe:
1. Скачайте последний релиз
2. Поместите key.json рядом с exe файлом
3. Запустите программу
## 🎮 Управление
- F1 - скрыть/показать окно захвата
- F2 - скрыть/показать окно перевода
- F3 - выход из программы
- T - перевести выделенный текст
## 🔧 Требования
- Python 3.8+
- Tesseract OCR
- Доступ в интернет
- Сервисный аккаунт Яндекс.Облака
